# Future Features

This file tracks optional capabilities that are not part of the v1 workflow yet.

## qmd

Potential role:

- local search over the wiki when the vault grows beyond what `wiki/index.md` and direct file reads handle comfortably
- faster retrieval across many wiki pages
- possible future replacement or supplement for simple manual page lookup

Current status:

- deferred
- do not depend on `qmd` for normal ingest or query workflows yet

Adoption trigger:

- revisit when the wiki contains enough pages and sources that manual index-driven lookup becomes slow or unreliable
