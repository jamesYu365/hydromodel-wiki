# Overview

This wiki is a research workspace maintained by Codex. Raw sources live in `raw/`. Synthesized knowledge lives in `wiki/`.

## How To Use This Wiki

1. Add markdown or PDF sources to `raw/`.
2. Ask Codex to ingest a specific source.
3. Review the created or updated pages.
4. Ask questions against the wiki.
5. Save durable answers back into the wiki as synthesis pages.
